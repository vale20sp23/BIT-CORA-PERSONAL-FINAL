Proyecto Grupal

Ahora te hablaré sobre el proyecto grupal enfocado en resolver una ODS

El problema que definimos fue el desinterés educativo en el Perú.

Para resolver esta problemátiva diseñamos HISTORIANDO,

un prototipo educativo para niños de 10 a 16 años. Utiliza una plataforma con sensores que reconoce objetos asociados a figuras históricas. Conectada a una laptop, desencadena audios personalizados al reconocer los objetos, transportando a los usuarios a diferentes épocas.

La base de "Historiando" está equipada con sensores de reconocimiento que pueden identificar objetos específicos colocados sobre ella. Está conectada a una laptop que actúa como el centro de control y procesamiento. La laptop ejecuta un software especializado diseñado para el prototipo. Los niños pueden seleccionar entre una variedad de figuras históricas, como científicos famosos, líderes políticos, artistas destacados, etc. Cada figura histórica está asociada con un objeto específico que se coloca sobre la base. Cuando el usuario coloca el objeto asociado con un personaje histórico en la base, los sensores reconocen automáticamente el objeto y activan el sistema. Una vez que el objeto es reconocido, se escucha un audio que relata los datos más importantes sobre el personaje. Este simulador transporta al usuario a la época y entorno del personaje histórico seleccionado.
